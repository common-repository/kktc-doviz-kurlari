== Description ==

KKTCMB Resmi Döviz Kurlarını gösteren basit bir widget.
Bilgileri http://www.kktcmerkezbankasi.org/kur/gunluk.xml bu adresten alıyor.

== Installation ==

1. 'kktc-doviz' klasörünü, `/wp-content/plugins/` altına kopyalayın.
2. 'Eklentiler' menüsünden aktif hale getirin.
